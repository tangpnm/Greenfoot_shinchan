Your individual Greenfoot assignment must include the following
1. The flamingo as a main character
2. Ability of the main character to do something, such as, eating, jumping, flying, singing, shooting, etc
3. A button that starts the game (Game Start button)
4. A score board that is shown when the game is over
5. A button that restarts the game (Play Again button)
6. Sound effects

Flamingo graphics are provided as PNG files with thier name sorted with respect to the order of the flamingo's movement.
You must use provided graphic(s) to create your main character.
Using multiple frames is optional. That is, the flamingo can be a static graphic. However, it should not be immobile.
